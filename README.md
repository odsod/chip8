# CHIP-8

Go implementation of the [CHIP-8 VM][chip8] according to [Cowgod's Technical Reference][cowgod].

Roms downloaded from [Zophar's Domain][zophar].

## Dependencies

~~~
github.com/nsf/termbox-go
~~~

## Example usage

~~~
chip8 -rom roms/TETRIS
~~~
